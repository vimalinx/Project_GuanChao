# Homepage Motion Language Design

**Direction:** 内敛含蓄，但在节奏、层次、边缘光和镜头感上体现技术含量。

## 气质定义

- 动效像精密设备启动，不像营销活动页。
- 首要目标是稳和准，其次才是炫。
- 所有动态都应服务于结构、阅读和设备质感，而不是抢内容。

## 视觉原则

- 大图稳定：首屏大图只做轻微推进、蒙版呼吸和扫光，不做明显位移。
- 信息分层：标题、说明、按钮、标签按短间隔递进出现。
- 卡片克制：卡片不跳、不弹，用边线增强、扫光和轻微镜头推进表达反馈。
- 区块 reveal：采用短位移、轻模糊恢复和裁切展开，而不是飞入。
- 科技底纹：保留白底企业感，在局部加入极轻的工业网格、线性扫描和冷色高光。

## 节奏规则

- 快反馈：`180ms`
- 常规切换：`320ms`
- 首屏/大图：`600ms`
- 默认 easing 使用偏硬朗的曲线，不用弹性或果冻效果。

## 首页三层实现

### 1. 首屏

- 设备海报保持完整展示。
- 图片初始有非常轻微的镜头推进，随后进入缓慢漂移。
- 蒙版和边框做低频呼吸。
- 标题、文案、按钮、应用标签分层进入。

### 2. 区块进入

- 模块进入采用短距离上移 + 清晰度恢复 + 裁切展开。
- 首屏以下的 section 使用 IntersectionObserver 触发一次性 reveal。
- 对首屏以上元素不做隐藏，避免闪烁。

### 3. 卡片与交互

- 产品卡、案例卡、新闻卡 hover 不上浮跳动。
- 图片只做 `1.02` 到 `1.04` 的轻微放大。
- 交互重点放在边线、角部光、扫光、标题颜色切换。
- 产品 tab 切换时卡片重新以短级联动画进入。

## 边界

- 不做满屏粒子、明显 3D、重玻璃、过强阴影。
- 不做夸张视差。
- 不做强蓝紫互联网风。
- 保持制造业企业站的可信度优先于“酷炫感”。
